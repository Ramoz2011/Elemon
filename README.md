# Elemon — an elemental battle game in Python

A Pokémon-inspired battle game where you fight using the elements themselves as your monsters.

![Demo](demo.gif)

## About

Elemon is a turn-based battle game where elements (like fire and water) act as your fighters, each with their own moves. You battle an opponent by choosing attacks until one side runs out of health. We built it as a two-person team project.

## How to play

1. Start the game
2. Take turns picking attacks against the opponent or switch
3. Reduce the opponent's health to zero to win

## How to run

1. Install the CMU Graphics library:
   ```
   pip install cmu-graphics
   ```
2. Download or clone this repo
3. Run the game:
   ```
   python_elemon.py
   ```

## Built with

- Python 3
- CMU Graphics (cmu_graphics)
